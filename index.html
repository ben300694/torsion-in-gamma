Torsion in (Gamma pi_2 K) / pi_1 K

This code was written for the bachelor thesis in mathematics with the topic "Torsion in $\Gamma(\pi_2 K)/\pi_1 K$".

Author: Benjamin Ruppik (University of Bonn)

Date: June 8th, 2016

Supervisors: Dr. Daniel Kasprowski and Prof. Dr. Peter Teichner (Max Planck Institute for Mathematics in Bonn)

This is the Git-repository for publishing the bachelor thesis and the Sage-code.

Instructions

The relevant Sage code is in the directory src/.

Use it by starting Sage in command line mode, then load the main file via attach('main.sage').
